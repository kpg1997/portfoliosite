Personal Project
Beargram
제작기간 : 21.03.08 ~ 21.03.24(17일)
개요 : 자신의 일상을 공유할 수 있게 만든 사이트입니다.
구현 기능 : 회원가입, 로그인·로그아웃, 아이디·비번 찾기, 게시글 작성·수정·삭제·목록·자세히 보기, 댓글 기능, 검색 기능, 좋아요 기능, 팔로우 기능, 개인 정보 수정, 채팅 기능
구현기능 상세히보기
- express-generator를 이용하여 프로젝트 생성.

- 회원가입
   - 아이디 생성 시 ajax를 이용하여 아이디 중복체크.
   - 비밀번호는 crypto를 이용하여 비밀번호 암호화.
   - 회원가입은 유효성 검사를 통해 정확히 입력해야 회원가입 가능.
   - 회원가입 성공 시 입력한 정보 DB 저장.
   - 회원가입 성공 시 회원가입 한 유저의 아이디로 폴더 생성.

- 로그인
   - 회원가입 시 사용한 아이디가 있는지 먼저 찾고 DB에 저장된 비밀번호를 복호화하여 입력한
      비밀번호와 같다면 로그인하고 session으로 유지.

- 메인페이지
   - 메인은 사용자가 팔로우한 유저의 게시글 출력.
   - Header는 jquery를 이용하여 header를 load하여 가져와 고정.
   - Header는 순서대로 홈, 사용자 게시글 목록, 사용자의 팔로우·팔로워, 개인 정보 수정,
      채팅, 로그아웃 순.

- 아이디 찾기 및 비밀번호 찾기
   - 필요로 하는 정보를 정확히 입력하면 사용자의 정보를 찾아 제공.
   - 비밀번호 찾기에 정보를 정확히 입력하면 비밀번호 변경할 수 있음.

- 게시글 작성·수정·삭제·목록·자세히 보기, 댓글 기능, 검색 기능, 좋아요 기능
   - 찾으려는 유저의 아이디, 이름으로 유저 검색. 게시글 내용으로 게시글 검색.
   - 사용자 검색 후 사용자 클릭 시 해당 사용자의 게시글 목록으로 이동.
   - 게시글 작성 시 파일 업로드는 multer를 이용하여 해당 유저 폴더에 파일을 업로드.
   - 게시글 수정·삭제는 해당 게시글의 작성자만 수정·삭제 가능.
   - 게시글 클릭 시 클릭한 게시글이 보이고 좋아요 버튼과 댓글을 작성할 수 있는 입력 부분과
      댓글 목록이 보임.
   - 댓글 작성자를 클릭 시 해당 유저가 작성한 게시글 목록으로 이동.
   - 좋아요 클릭 시 ajax를 이용하여 해당 게시글의 좋아요 수를 출력.

- 팔로우 기능
   - 사용자를 팔로우한 사람들의 목록과 사용자가 팔로우한 목록.
   - 다른 사용자 게시글 목록에서 팔로우 버튼을 누를 시 팔로우 및 언팔로우.
   - ajax를 이용하여 사용자의 팔로우·팔로워의 수를 가져와 출력.

- 개인 정보 수정
   - 사용자의 비밀번호를 정확히 입력하면 개인정보 수정페이지로 이동.
   - 사용자의 프로필 이미지, 이름, 비밀번호, 전화번호, 이메일 수정.
   - 비밀번호 변경 시 로그인페이지로 이동.

- 채팅 기능
   - 채팅버튼을 누른 유저들의 목록을 출력.
   - ajax로 이용하여 채팅 및 상대방 채팅을 출력.

- 로그아웃
   - 로그아웃 시 로그인 페이지 이동.
   - session을 destroy하여 로그아웃.
