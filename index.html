89Namu
제작기간 : 2021.08.15 ~ 2021.08.31(16일)/ 제작인원 : 4명
개요 : Ethereum으로 물건을 구입하면 토큰(Holomi)으로 페이백 해드립니다 Holomi로 나만의 나무(HolomiTree)를 구입하여 녹색지구를 만들기 위한 프로젝트입니다.
내가 구현한 기능
         - Frontend+Backend : 제품 구입(Shop)
         - Solidity : deposit, withdraw, buyToken, cancellationDeposit 제작
구현기능 상세히보기
기술문서
- Backend
   - MariaDB에 productList에 저장되어 있는 목록들을 보내줌.
   - 물품 구매 시 MariaDB를 이용하여 사용자 아이디, 구매한 물품이름, 가격 저장.

- Frontend
   - 물건 구매는 로그인을 해야만 구매를 진행할 수 있다.
   - PalguNamu.sol 안에 제작한 buying 함수를 이용하여 진행하고 redux-saga를 이용하여 비동기 진행

- Solidity
   - deposit
       : isDeposited를 이용하여 deposit인지 withdraw 상태인지 확인하여 deposit 진행

   - withdraw:
       : isDeposited를 이용하여 deposit인지 withdraw 상태인지 확인하여 withdraw 진행

   - buyToken
       : 자기가 가지고 있는 Ethereum으로 Holomi 토큰을 구입

   - cancellationDeposit
       : deposit인 상태였을 때 강제적으로 취소하여 deposit을 정지 시킬 수 있다.

BearGram 구현 기능 및 기술문서


회원가입


   - 아이디 생성 시 ajax를 이용하여 아이디 중복체크.
   - 비밀번호는 crypto를 이용하여 비밀번호 암호화.
   - 회원가입은 유효성 검사를 통해 정확히 입력해야 회원가입 가능.
   - 회원가입 성공 시 입력한 정보 DB 저장.
   - 회원가입 성공 시 회원가입 한 유저의 아이디로 폴더 생성.

로그인


   - 회원가입 시 사용한 아이디가 있는지 먼저 찾고 DB에 저장된 비밀번호를 복호화하여 입력한
      비밀번호와 같다면 로그인하고 session으로 유지.

메인페이지


   - 메인은 사용자가 팔로우한 유저의 게시글 출력.
   - Header는 jquery를 이용하여 header를 load하여 가져와 고정.
   - Header는 순서대로 홈, 사용자 게시글 목록, 사용자의 팔로우·팔로워, 개인 정보 수정,
      채팅, 로그아웃 순.

아이디 찾기 및 비밀번호 찾기


   - 필요로 하는 정보를 정확히 입력하면 사용자의 정보를 찾아 제공.
   - 비밀번호 찾기에 정보를 정확히 입력하면 비밀번호 변경할 수 있음.


게시글 작성·수정·삭제·목록·자세히 보기, 댓글 기능, 검색 기능, 좋아요 기능

기술 문서

오점뭐 구현 기능


- Backend
   - MongoDB를 이용하여 저장
      - post : 게시판 생성.
      - get : 게시판 목록 가져오기, 게시판 자세히 보기.
      - put : 게시판 업데이트.
      - delete : 게시판 삭제.

- Frontend
   - 게시판 목록 : MongoDB에 저장되어 있는 게시판 목록 가져오기.
   - 게시판 작성 : 게시판 작성 시 로그인 유무 확인하여 진행.
   - 게시판 수정 : 로그인 된 아이디가 작성된 유저의 아이디와 같으면 수정 가능.
   - 게시판 삭제 : 로그인 된 아이디가 작성된 유저의 아이디와 같으면 삭제 가능.

오점뭐 기술문서


Main Page

- 한식, 중식, 일식, 양식의 메뉴판이 있으며 클릭 시 Book Page로 이동.

Login / SignUp Page

- 회원가입 시 MongoDB를 이용하여 회원정보 저장.
- 로그인 시 cookie와 session에 저장.

Book Page

- 책 마우스 커서를 올리면 책이 펼쳐지며 메뉴가 랜덤으로 나옴.
- 댓글 작성 및 최대 댓글 5개까지 볼수 있음.
- 메뉴판 클릭 시 카카오 맵으로 이동.

FreeBoard Page

- 자유롭게 작성 할 수 있는 페이지.
- 게시판 목록, 작성, 삭제, 수정 기능이 있음.
- 게시판 클릭 시 조회수가 올라가며 댓글 작성, 댓글 목록을 볼 수 있음.

AddBoard Page

- 원하는 메뉴를 추가 시켜달라고 작성 할 수 있는 페이지.

KaKaoMap Page

- 카카오맵 API를 이용하여 검색 시 원하는 곳 검색 가능.

Hosting

- Oracle Cloud를 이용하여 호스팅 진행
